C4D Mega Menu will automatically convert your existing menu or menus into a mega menu. You can then add any WordPress widget to your menu, restyle your menu using the theme editor and change the menu using the built in settings. Max Mega Menu is a complete menu management plugin, perfect for taking control of your existing menu and turning it into a user-friendly, accessible and touch ready menu with just a few clicks.

== Description ==

A simple and lighweight plugin for creating mega menu.

You can set a menu item to mega mode in Menu manager.

A sidebar will be auto created and insert to this menu item. 

Than you can insert widget to this sidebar.

To enable options page, please install and active C4D Plugin Manager and Redux Framework

https://wordpress.org/plugins/c4d-plugin-manager/

https://wordpress.org/plugins/redux-framework/

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-mega-menu` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
